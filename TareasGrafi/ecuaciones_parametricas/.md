## Práctica 5: Generación de Figuras mediante Ecuaciones Paramétricas con OpenCV y NumPy

---
## Objetivo

Generar figuras geométricas utilizando ecuaciones paramétricas, donde cada punto calculado mediante funciones matemáticas se representa como un píxel dentro de una matriz, y visualizar el resultado utilizando OpenCV.

---
## Marco Teórico

Las ecuaciones paramétricas son ecuaciones matemáticas que permiten representar curvas mediante el uso de una variable adicional llamada parámetro, generalmente representada como **t**.
En lugar de expresar una relación directa entre x e y, se utilizan dos ecuaciones:
- x = f(t)
- y = g(t)

Esto permite representar curvas complejas como:
- Círculos
- Elipses
- Espirales
- Flores matemáticas
- Curvas tipo infinito

Las funciones trigonométricas como seno (sin) y coseno (cos) son fundamentales para generar estas figuras.
Una imagen digital está formada por una matriz de píxeles. Cada píxel tiene un valor entre:
- 0 representa negro
- 255 representa blanco
- Valores intermedios representan tonos de gris

NumPy permite realizar cálculos matemáticos y crear matrices, mientras que OpenCV permite visualizar la imagen generada.

---
## Código 

```python
# Importa la librería NumPy para realizar cálculos matemáticos
import numpy as np

# Importa la librería OpenCV para mostrar imágenes
import cv2 as cv

# Crea una matriz de 600x600 píxeles llena de ceros
# Cada cero representa un píxel negro
# dtype=np.uint8 permite valores entre 0 y 255
img = np.zeros((600,600), dtype=np.uint8)


# Define el centro de la imagen
# Esto permite que las figuras aparezcan en el centro
centro_x = 300
centro_y = 300

# Genera valores desde 0 hasta 2π
# linspace crea 1000 valores igualmente espaciados
# Esto permite que las curvas sean suaves
t = np.linspace(0, 2*np.pi, 1000)

# ECUACIÓN PARAMÉTRICA DEL CÍRCULO
# x = r cos(t)
# y = r sin(t)
# Define el radio del círculo
radio = 100
# Recorre todos los valores del parámetro t
for i in range(len(t)):
    # Calcula la coordenada x
    x = int(centro_x + radio*np.cos(t[i]))
    # Calcula la coordenada y
    y = int(centro_y + radio*np.sin(t[i]))
    # Asigna el valor 255 al píxel
    # Esto representa el color blanco
    img[y,x] = 255


# ECUACIÓN PARAMÉTRICA DE LA ELIPSE
# x = a cos(t)
# y = b sin(t)
# Define los valores de los ejes
a = 150
b = 50
for i in range(len(t)):
    x = int(centro_x + a*np.cos(t[i]))
    y = int(centro_y + b*np.sin(t[i]))
    img[y,x] = 200


# ECUACIÓN PARAMÉTRICA DE LA ESPIRAL
# x = t cos(t)
# y = t sin(t)
for i in range(len(t)):
    x = int(centro_x + (t[i]*50)*np.cos(t[i]))
    y = int(centro_y + (t[i]*50)*np.sin(t[i]))
    # Verifica que el punto esté dentro de la imagen
    if 0 <= x < 600 and 0 <= y < 600:
        img[y,x] = 150


# ECUACIÓN PARAMÉTRICA DE LA ROSA
# x = r cos(4t) cos(t)
# y = r cos(4t) sin(t)
for i in range(len(t)):
    r = 100*np.cos(4*t[i])
    x = int(centro_x + r*np.cos(t[i]))
    y = int(centro_y + r*np.sin(t[i]))
    if 0 <= x < 600 and 0 <= y < 600:
        img[y,x] = 255


# ECUACIÓN PARAMÉTRICA DE LA LEMNISCATA
# x = a cos(t)/(1+sin²(t))
# y = a sin(t)cos(t)/(1+sin²(t))
# Define el tamaño de la curva
a = 200
for i in range(len(t)):
    x = int(centro_x + (a*np.cos(t[i]))/(1+np.sin(t[i])**2))
    y = int(centro_y + (a*np.sin(t[i])*np.cos(t[i]))/(1+np.sin(t[i])**2))
    if 0 <= x < 600 and 0 <= y < 600:
        img[y,x] = 255


# Muestra la imagen en una ventana
cv.imshow("Ecuaciones Parametricas", img)

# Espera hasta que el usuario presione una tecla
cv.waitKey(0)

# Cierra todas las ventanas
cv.destroyAllWindows()
