## Práctica 3: Transformaciones Geométricas con OpenCV y NumPy
 
---
## Objetivo

Aplicar transformaciones geométricas a una imagen utilizando matrices y funciones de OpenCV, para modificar su posición, tamaño, orientación y forma.
---

## Marco Teórico

Las transformaciones geométricas permiten modificar la posición de los píxeles en una imagen mediante operaciones matemáticas.

Cada píxel tiene coordenadas (x, y), y estas coordenadas pueden modificarse usando matrices.

Principales transformaciones geométricas:

- Traslación: mueve la imagen
- Rotación: gira la imagen
- Escalado: cambia el tamaño
- Cizallamiento: inclina la imagen
- Reflexión: invierte la imagen

OpenCV permite aplicar estas transformaciones mediante funciones optimizadas.

---
## Código 

```python
# Importa la librería NumPy
# Se usa para crear matrices que representan imágenes
import numpy as np


# Importa la librería OpenCV
# Permite procesar imágenes y aplicar transformaciones
import cv2 as cv


# Crea una imagen negra de tamaño 300x300 pixeles
# np.zeros crea una matriz llena de ceros
# Cada cero representa un píxel negro
# dtype=np.uint8 permite valores entre 0 y 255
img = np.zeros((300,300), dtype=np.uint8)


# Crea un cuadrado blanco dentro de la imagen
# Se modifican las filas 100 a 200 y columnas 100 a 200
# El valor 255 representa el color blanco
img[100:200,100:200] = 255


# Obtiene el número de filas y columnas de la imagen
# shape devuelve el tamaño de la matriz
filas, columnas = img.shape


## TRASLACION

# Crea la matriz de traslación
# Esta matriz moverá la imagen 50 pixeles a la derecha y 50 hacia abajo
M_traslacion = np.float32([[1,0,50],[0,1,50]])


# Aplica la transformación de traslación
# warpAffine mueve la imagen usando la matriz de transformación
traslacion = cv.warpAffine(img, M_traslacion, (columnas, filas))


## ROTACION

# Crea la matriz de rotación
# (columnas/2, filas/2) es el centro de la imagen
# 45 es el ángulo en grados
# 1 es el factor de escala
M_rotacion = cv.getRotationMatrix2D((columnas/2, filas/2), 45, 1)


# Aplica la rotación a la imagen
rotacion = cv.warpAffine(img, M_rotacion, (columnas, filas))


## ESCALADO

# Cambia el tamaño de la imagen
# fx=1.5 aumenta el tamaño horizontal
# fy=1.5 aumenta el tamaño vertical
# INTER_NEAREST mantiene la forma de los píxeles
escalado = cv.resize(img, None, fx=1.5, fy=1.5, interpolation=cv.INTER_NEAREST)


## CIZALLAMIENTO

# Crea la matriz de cizallamiento
# Esta matriz inclina la imagen
M_cizalla = np.float32([[1,0.5,0],[0,1,0]])


# Aplica el cizallamiento a la imagen
cizalla = cv.warpAffine(img, M_cizalla, (columnas, filas))


## REFLEXION

# Aplica una reflexión horizontal
# flip invierte la imagen como un espejo
reflexion = cv.flip(img, 1)


## MOSTRAR RESULTADOS

# Muestra la imagen original
cv.imshow("Imagen Original", img)

# Muestra la imagen trasladada
cv.imshow("Traslacion", traslacion)

# Muestra la imagen rotada
cv.imshow("Rotacion", rotacion)

# Muestra la imagen escalada
cv.imshow("Escalado", escalado)

# Muestra la imagen inclinada
cv.imshow("Cizallamiento", cizalla)

# Muestra la imagen reflejada
cv.imshow("Reflexion", reflexion)

# Espera hasta que el usuario presione una tecla
cv.waitKey(0)

# Cierra todas las ventanas abiertas
cv.destroyAllWindows()
