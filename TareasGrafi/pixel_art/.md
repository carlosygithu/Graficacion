# Generación de Pixel Art usando OpenCV

## Objetivo

Generar una imagen tipo pixel art utilizando una matriz de enteros en el rango de 0 a 255.

---

## Descripción

Una imagen digital puede representarse como una matriz de valores entre 0 y 255, donde:

- 0 = negro
- 255 = blanco
- Valores intermedios = tonos de gris

---

## Código

```python
import numpy as np
import cv2

imagen = np.array([
    [0, 255, 0, 255],
    [255, 0, 255, 0],
    [0, 255, 0, 255],
    [255, 0, 255, 0]
], dtype=np.uint8)

imagen = cv2.resize(imagen, (400,400), interpolation=cv2.INTER_NEAREST)

cv2.imwrite("resultado.png", imagen)
cv2.imshow("Pixel Art", imagen)
cv2.waitKey(0)
cv2.destroyAllWindows()
```
