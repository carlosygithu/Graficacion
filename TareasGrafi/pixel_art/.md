## Práctica 1: Generación de Imagen Pixel Art con OpenCV y NumPy
 
---
## Objetivo
Generar una imagen tipo Pixel Art utilizando una matriz de enteros en el rango de 0 a 255, donde cada valor representa la intensidad de un píxel, y visualizarla utilizando OpenCV.
---
## Marco Teórico
Una imagen digital está formada por una matriz de píxeles. Cada píxel tiene un valor numérico que representa su intensidad.

En escala de grises:
- 0 representa negro
- 255 representa blanco
- Valores intermedios representan tonos de gris

La librería NumPy permite crear y manipular matrices, mientras que OpenCV permite visualizar imágenes.
El Pixel Art es una técnica donde las imágenes se crean manipulando directamente los píxeles.
---

## Código

```python
# Importa la librería NumPy para trabajar con matrices
import numpy as np

# Importa la librería OpenCV para procesamiento de imágenes
import cv2 as cv


# Crea una matriz de 10x10 llena de ceros
# Cada cero representa un píxel negro
# dtype=np.uint8 permite valores entre 0 y 255
img = np.zeros((10,10), dtype=np.uint8)


# Asigna el valor 255 a ciertos píxeles
# El valor 255 representa el color blanco

# Pixel en fila 2, columna 2
img[2,2] = 255

# Pixel en fila 2, columna 7
img[2,7] = 255

# Pixel en fila 5, columna 4
img[5,4] = 255

# Pixel en fila 7, columna 7
img[7,7] = 255


# Redimensiona la imagen a 300x300 píxeles
# INTER_NEAREST mantiene la forma cuadrada de los píxeles
img_grande = cv.resize(img, (300,300), interpolation=cv.INTER_NEAREST)


# Muestra la imagen en una ventana
cv.imshow("Pixel Art", img_grande)


# Espera hasta que el usuario presione una tecla
cv.waitKey(0)


# Cierra todas las ventanas
cv.destroyAllWindows()
